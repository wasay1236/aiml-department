# Breast Cancer Detection Using Machine Learning

## 📖 Overview
This project uses **Machine Learning (SVM)** to classify breast tumors as **benign or malignant**, supporting early detection.

<img src="https://cdn-images-1.medium.com/max/2600/1*gNcFEL1cpGpDC4vo1zUAWA.png" />

---

## 🧾 Project Description
This project implements a **classification model using Support Vector Machine (SVM)** on the Breast Cancer Wisconsin dataset.  
The system analyzes input features such as radius, texture, and area to predict whether a tumor is **cancerous or non-cancerous**.  

The model is trained, tested, and optimized using **feature scaling and hyperparameter tuning**, achieving high accuracy and demonstrating the practical use of AI in healthcare.

---

## 🎯 Objectives
- Build a prediction model  
- Analyze medical dataset  
- Achieve high accuracy

## 📊 Dataset

## 🤖 Model
- Algorithm: **Support Vector Machine (SVM)**  
- Accuracy: **~97% (after optimization)**
