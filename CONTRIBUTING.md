## ⚠️ Before You Start

- Screenshots shown in guides are for **demonstration purposes only**
- Each department has its **own repository**
- ❗ Upload your project **ONLY in your respective department repository**
- Follow all contribution guidelines strictly

---

## 🚀 Contribution Workflow

Follow this process:
`Fork → Make Changes → Commit → Create Pull Request`

---

## 📂 Project Submission Guidelines

Navigate to:
`Project → batch-2022-2026`

### ✅ Requirements:
- Create a folder with your **project name**
- Maintain clean folder structure
- Include all required files
- Avoid unnecessary or duplicate files

---

## 🏷️ Pull Request Guidelines

### 📌 Title Format (MANDATORY)
`<Project Name> | <Roll Number>`

### ✅ Example:
`AI Chatbot for Students | 22A51A0001`

---

### 📄 Description Must Include:
- Project Title  
- Roll Number  
- Team Members (Full Names)

#### 💡 Example:
`AI Chatbot for Students | 22A51A0001`

`Team Members:`

`John Doe`

`Jane Doe`


---

### ✅ Good PR Practices

- Keep PRs **clean and focused**
- Upload only relevant files
- Follow naming conventions
- Ensure your project works properly

---

## 🐛 Issue Guidelines

If you are creating an issue:

### ✅ Include:
- Clear title  
- Proper description  
- Steps to reproduce (if bug)  
- Expected vs actual behavior  

### ❌ Avoid:
- Vague issues  
- Duplicate issues  
- Irrelevant content  

---

## 🤝 Code of Conduct

We expect all contributors to:

- Be respectful and professional  
- Communicate clearly  
- Help others learn and grow  
- Avoid spam or inappropriate behavior  

Any violation may result in rejection of contributions.

---

## 🔍 Review Process

- All PRs are reviewed by maintainers  
- Automated checks must pass  
- Maintainers may request changes  
- Only valid submissions will be merged  

---

## 🏷️ Labels Used

| Label | Meaning |
|------|--------|
| project-submitted | PR created |
| project-under-review | Under review |
| needs correction | Fix required |

---

## ❌ Common Mistakes

- Wrong repository submission  
- Incorrect PR title format  
- Missing files  
- Poor structure  
- Editing unrelated files
